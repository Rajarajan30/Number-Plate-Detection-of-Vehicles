# Number-Plate-Detection-of-Vehicles & OCR

A real-time Number Plate Detection and OCR system using YOLO for detection and EasyOCR/Tesseract for text extraction, wrapped in a Streamlit web app.

## The primary goals of this system include: 

- Detecting number plates from images of vehicles captured in different 
environments and lighting conditions. 
- Extracting number plate regions accurately while handling challenges like 
occlusions, varying backgrounds, and different plate formats. 
- Applying Optical Character Recognition (OCR) to extract the alphanumeric text 
from the detected plates. 
- Deploying the system as a user-friendly web application for real-time processing.

## üîÑ Workflow

1. **Data Collection**
   - Acquiring images of vehicles from various environments, including 
different angles, lighting conditions.

2. **Preprocessing**
   - Image resizing, normalization and Augmentation techniques.

3. **Model Training**
   - Using the YOLO (You Only Look Once) object detection framework for 
efficient real-time number plate detection.Training the model using GPU acceleration to optimize performance.

4. **OCR Extraction**
   - EasyOCR + Tesseract extract plate text.

5. **Post-processing**
   - Cleaning and formatting text.

6. **Deployment**
   - Display cropped plate, detected text by using Streamlit-based web application.
